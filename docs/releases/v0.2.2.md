# v0.2.2

`v0.2.2` hardens the `v0.2.x` runtime for real MCP usage and repeatable
release checks.

## Highlights

- Added `gwt-context-mcp-smoke`, a real stdio MCP client smoke that launches
  `python -m gwt_context`, calls public tools, and reads `gwt://attention/last`.
- Documented Codex MCP setup with `codex mcp add gwt-context`.
- Added an infrastructure-owned OpenAI-compatible external subscriber adapter
  that emits JSON broadcast proposals without coupling provider code to
  `application`.
- Added `scripts/release_gate.py` for local pre-release verification and
  `scripts/qwen_sanity.py` for bounded Qwen/OpenAI-compatible sanity runs.
- Extended CI with runtime smokes and explicit boundary checks.
- Improved benchmark trace HTML with proposal grouping, inhibited proposal
  rationale, and workspace-change summaries.

python scripts/release_gate.py
