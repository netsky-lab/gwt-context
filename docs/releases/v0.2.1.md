# v0.2.1

`v0.2.1` is a polish release focused on making the `v0.2.0` runtime easier to
demo, inspect, and extend.

## Highlights

- Added reproducible local demo scenarios for graph attention, collection
  resolution, and contradiction-aware bus traces.
- Added a real in-process MCP usage loop that exercises store, attend, bus
  inspection, trace explanation, and workspace inspection with hash embeddings.
- Added an external subscriber proof-of-concept showing how an adapter-owned
  agent loop can emit sanitized bus proposals without provider SDK coupling.
- Improved benchmark trace HTML with phase timelines, bus action counts, policy
  skip counts, and subscriber status badges.
- Updated quickstart and README links for the new demo and subscriber flows.
